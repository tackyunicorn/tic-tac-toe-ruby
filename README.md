# :hash: tic-tac-toe :hash:
:star: A tic-tac-toe game in ruby :star:

## Game
Implemented as a class with methods to give players turns and check for a winning configuration.  
It looks like this:

```
<=============PLAYER 1=============>

----------------
| O |  X |    |
----------------
| X |  X |  X |
----------------
| O |  O |    |
----------------

Player 1 wins!
```
